Welcome to my homepage!
Dustin Chen

This website is an email-themed personal website. The theme is inspired by a front-end coding challenge I got when I first started learning web development – I wasn’t skilled enough to complete the assignment back then, but now I can!

The three most significant features of this website:

1.) The main window for the email dynamically changes size (implemented using $(window).resize). This lets the email expand to fit different screen sizes, and it also scales with the web browser when it is resized.

2.) Clicking 3 different emails will make different emails pop up on the screen. Clicking the collapse button on the top left of the screen, in the nav bar, cycles between 3 different screen layouts. These two things were both done using the jquery .show and .hide commands along with .css to modify the positioning. Notice the main email window expands to fill the empty space when a menu is collapsed.

3.) Small pseudo-selector/clicking stuff. Hovering over the collapse button, the different inboxes, and different emails produces different shadows/borders/cursor effects. Clicking an inbox will make that inbox change to blue font and also produces a border to the left. Unfortunately I haven’t implemented the inbox filtering functionality yet.
